Hawaii Precipication and Tempurature API

Here are the endpoints:
  1. Precipication
  2. Stations
  3. Temperatures
  4. Date Range
    • Include a start date to see the min, max, and average temperature from that date on
    • Include another slash followed by a later date to see the min, max, and average temperature for all days between the given dates
    • Data is limited to 2010/01/01 through 2017/08/17
